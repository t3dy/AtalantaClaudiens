## Data Flow

```
data/emblem_manifest.json  (canonical identity — agents edit this)
        |
        v
seed_identity.py  -->  emblem_identity table (SQLite)
        |
        v
build_site.py  -->  site/emblems/emblem-NN.html (static HTML)
```

## How to Find an Emblem's Data

1. **Identity** (number, image, confirmation): `data/emblem_manifest.json`
2. **Content** (motto, epigram, discourse, analysis): `SELECT * FROM emblems WHERE number = N`
3. **Scholarly refs**: `SELECT * FROM scholarly_refs sr JOIN emblems e ON sr.emblem_id = e.id WHERE e.number = N`
4. **Source authorities**: `SELECT sa.name FROM emblem_sources es JOIN source_authorities sa ON es.authority_id = sa.id JOIN emblems e ON es.emblem_id = e.id WHERE e.number = N`
5. **Dictionary terms**: `SELECT dt.label FROM term_emblem_refs ter JOIN dictionary_terms dt ON ter.term_id = dt.id JOIN emblems e ON ter.emblem_id = e.id WHERE e.number = N`
6. **Detail page**: `site/emblems/emblem-{NN:02d}.html`
7. **Image file**: `site/images/emblems/emblem-{NN:02d}.jpg` (check manifest for confirmation)

## How to Add/Update Emblem Data

### Adding an image
1. Download the image to `site/images/emblems/emblem-{NN:02d}.jpg`
2. Update `data/emblem_manifest.json`: set `image_file`, `image_confirmed: true`, `image_source`
3. Run `python scripts/seed_identity.py` to sync to DB
4. Run `python scripts/build_site.py` to regenerate HTML

### Adding scholarly content
1. Insert into `scholarly_refs` table via extraction script or seed data
2. Run `python scripts/seed_emblem_analyses.py` to regenerate analysis blocks
3. Run `python scripts/build_site.py` to regenerate HTML

### Adding dictionary term links
1. Insert into `term_emblem_refs` table
2. Run `python scripts/seed_emblem_analyses.py` (analysis blocks include term links)
3. Run `python scripts/build_site.py`

## Emblem Page Structure (HTML)

Each emblem page at `site/emblems/emblem-NN.html` contains:
- **Nav**: Previous / Gallery / Next links
- **Title**: "Emblem {Roman} — {Label}"
- **Source Panel** (left 60%):
  - Plate image (or placeholder)
  - Motto block
  - Epigram (collapsible)
  - Discourse summary
  - Analysis block (AI-assembled: overview, sources, significance, terms)
  - Alchemical stage badge
- **Scholarship Panel** (right 40%):
  - Scholarly commentary cards (author + confidence + summary)
  - Maier's source links

## Cross-References

Each emblem connects to:
- **Source authorities** via `emblem_sources` → links to `/sources.html#{authority_id}`
- **Dictionary terms** via `term_emblem_refs` → links to `/dictionary/{slug}.html`
- **Scholars** via `scholarly_refs` → linked to bibliography
- **F&F** via manifest `furnace_fugue_url` → external reference
