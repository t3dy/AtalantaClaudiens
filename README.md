# AtalantaClaudiens

A digital humanities website showcasing H.M.E. De Jong's scholarship on Michael Maier's *Atalanta Fugiens* (1618) — an alchemical emblem book combining 50 engraved plates, Latin mottos, epigrams, prose discourses, and three-voice musical fugues.

## What This Project Does

- Presents all 50 emblems with comparative views: original source material alongside scholarly commentary
- Cross-references scholarship from De Jong, Tilton, Craven, Wescott, Pagel, Miner, and others
- Maps Maier's textual sources (Turba Philosophorum, Rosarium, Tabula Smaragdina, Ovid, etc.) to individual emblems
- Provides an alchemical dictionary, reception timeline, and bibliography
- Tracks data provenance: every datum marked by source method, review status, and confidence level

## Architecture

```
Source Corpus (.md) → Python Scripts → SQLite → build_site.py → Static HTML → GitHub Pages
```

No frameworks. No build tools. No runtime dependencies. Pure HTML/CSS/JS served from GitHub Pages.

## Rebuild

```bash
python scripts/init_db.py
python scripts/seed_from_json.py
python scripts/extract_dejong.py
python scripts/extract_secondary.py
python scripts/seed_dictionary.py
python scripts/seed_timeline.py
python scripts/build_site.py
python scripts/validate.py
```

## Data Provenance

| Source Method | Meaning |
|--------------|---------|
| DETERMINISTIC | Regex/formula/filename parsing |
| CORPUS_EXTRACTION | Parsed from source text |
| LLM_ASSISTED | Generated by language model |
| SEED_DATA | Loaded from seed JSON |
| HUMAN_VERIFIED | Manually confirmed |

All LLM-assisted content is marked with review badges on the site.
